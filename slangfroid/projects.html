Another instance of a project that i really enjoyed working on is pictured on the left. It is an example of a scatterplot matrix that I used to experiment with machine learning data. The blue dots are all true data points and the orange ones are false. This particular scatterplot was made with seaborn and then I used sklearn models to train and test the data. Overall for the true data I was able to achieve 92% for the true data, and 93% for the recall. For the false data I achieved 88% precision and a 86% recall. My overall accuracy was 90%. That translates to 276 false negatives, and 226 false positives. This graph is just a sample of what I have done with machine learning and I have big plans for what I am going to make in the future.
